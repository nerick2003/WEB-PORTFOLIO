# Images Folder

Place your profile picture here.

## Profile Picture

1. **File Name**: Name your image file (e.g., `profile.jpg`, `profile.png`, `my-photo.jpg`)
2. **Update Path**: Edit `config/data.js` and update the `profileImage` path in `personalInfo`:
   ```javascript
   profileImage: "images/your-filename.jpg"
   ```
3. **Recommended Size**: 400x400 pixels or larger (square images work best)
4. **File Formats**: JPG, PNG, or WebP

## Example

If your image is named `profile.jpg`, make sure `config/data.js` has:
```javascript
profileImage: "images/profile.jpg"
```
